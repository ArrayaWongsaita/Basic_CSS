นี่คือตัวอย่างการใช้ float: left; ภาพจะลอยอยู่ทางซ้ายของข้อความ ข้อความจะไหลไปอยู่รอบๆ รูปภาพตามธรรมชาติ เหมือนเวลาอ่านหนังสือพิมพ์ที่มีรูปกับบทความ เทคนิคนี้เคยนิยมใช้จัด layout แต่ปัจจุบันนิยมใช้ Flexbox หรือ CSS Grid แทน

แต่ float ก็ยังมีประโยชน์ในบางกรณี เช่น จัดรูปประกอบในบทความ หรือทำปุ่มลอยข้างๆ บทความให้ผู้ใช้คลิกได้สะดวก
